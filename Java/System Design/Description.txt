Program begins in Controller.

It instantiates Dashboard, passing it a reference to Model. The program then runs entirely in Dashboard, calling methods in Model as appropriate.

Dashboard: 
*A text box to type in a roll, click roll, then see the results.
*NPC generator.	
	> Save To File (w/ header) in NPCs.txt
	> Save For Session (w/ header)
	> Edit Saved NPCs (via SN)
	> Delte Saved NPCs (via SN)
*Encounter XP Calculator.
*Encounter Generator.
*Monster Searcher.
